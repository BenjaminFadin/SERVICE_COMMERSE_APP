Everything for an MVP

Core user journeys are ready out of the box.

See flow
Client side
  • Search and filter providers by service, location, rating.
  • Select services, see durations and real-time slots.
  • Auth (email/password), profile, saved payment placeholder.
  • Booking review + confirmation / success state.
  • My bookings (upcoming, past, cancel, reschedule placeholder).
Provider side
  • Dashboard with KPIs, schedule density and reviews.
  • Booking list with status (new, confirmed, checked-in).
  • Service catalog: price, duration, category, visibility.
  • Team and resources placeholders (chair/room/staff).
  • Profile page with hours, socials and location preview.
Ops / comms
  • Notification placeholders (email/SMS/push ready).
  • Cancellation policy & no-show protection CTA.
  • Marketing CTA for promos and add-ons upsell.
  • Embeddable calendar widget vibe for partner sites.
  • Lightweight JS hooks for real API wiring later.
Booking flow

Click. Pick. Confirm.

  1. Discover services on marketplace.html.
  2. Open a provider page (business.html) to see services.
  3. Select a service and slot (booking.html).
  4. Review summary and confirm (booking-success.html).
  5. Manage on dashboard-client.html.
Provider flow

Open for business in minutes.

  1. Sign in / sign up (auth-login.html).
  2. Configure profile and hours (dashboard-provider.html).
  3. Add services, durations and pricing.
  4. Share booking link and watch slots fill.
  5. Track KPIs and reviews in the dashboard.
